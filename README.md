# BusinessSearchApp

I developed an app that let's you search ALL the buisnesses such as restaurants, clothing, museums around your place, divided to three sections: Expensive, Pricy, Cheap and present thier info!.
Just choose the food you would like to eat and MyFoodApp will do the rest!

Devolpoing process
I installed Axios and used Yelp API to fetch information about all the buisnesses around you.

Running Process
Add your Yelp API client ID and password in the yelp.js file. To change the search location go to useResults.js file and change line 20.
